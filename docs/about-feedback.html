Feedback and Issues

If you and idea on how to improve Headings, Landmarks and Links Side Panel or find a problem please let us know by creating an issue in the GitHib repository for Headings, Landmarks and Links Side Panel. Please provide the following information:

Github Issue (Preferred)

  1. Create Headings, Landmarks and Links Side Panel Github Issue
  2. If an issue or problem, include a URL to were the issue can be replicated.
  3. Include a short description of the issue or idea.
  4. How we can contact you for more information and verification the issue is resolved.

Note: Github issues are preferred since it provides a public record of the issue and allows other people to comment. The public record means the issue will remain until resolved, where as e-mails can sometimes missed.

E-mail

Send an e-mail to Jon Gunderson (jongund@illinois.edu) and include "Headings, Landmarks and Links Side Panel" in the subject line and the following information in the body if it is problem:

  1. If an issue or problem, include a URL to were the issue can be replicated.
  2. Include a short description of the issue or idea.
  3. How we can contact you for more information and verification the issue is resolved.
