Headings, Landmarks and Links Browser Extension

Browser Extension

Overview

The Headings, Landmarks and Links Side Panel (H2L) extension provides a browser side panel to view the headings, landmark regions and links on a web page. It can be used to verify the proper use of headings and landmarks in identifying the layout and structure of a web page and testing links t see if the link text provides a good description of the target of the link. The extension uses the the OpenA11y Evaluation Library to collect information the page title, headers, landmarks and links for viewing in the side panel.
