Options

Highlight

Option Description
Highlight follows focus
  • When enabled (default) the user moves focus between items in the heading, landmarks and links tabs, the associated item on the page will be scrolled into view and highlighted automatically.
  • Some people may want to disable automatic highlighting if the scolling is a distraction or disorientation.
  • When automatic highlighting is disabled, use the space or enter key to scoll the associated item into view.
Border Size Sets the width of the border used to highlight items on the web pages. The following sizes are available:
  • small
  • medium
  • large (default)
  • x-large
  • xx-large
Border Style The style used for the border. The following styles are available:
  • solid
  • dashed (default)
  • dotted

Landmark and Heading Filters

Option Description
Show hidden and off-screen headings.
  • Some authors add headings only intended for use by screen readers by making them very small or moving them off-screen.
  • Some screen readers detect these techniques and do not include the headings in their list of headings to reduce the number of heading users have to listen too.
  • The filter mimics screen readers in not showing the small and off-screen headings.
  • Default is disabled.
Show regions without accessible names
  • Authors often inadvertently create landmarks by using html elements like aside (e.g. complementary) or nav (e.g. navigation).
  • Screen readers filter out landmarks if there is more than one of the same type of landmark (e.g. complementary or navigation) and the landmark does not have an accessible name.
  • Default is disabled.

Link Filters

Option Description
Show internal links
  • Internal links scroll content on the current page into view.
  • When enabled internal links are included in the list of links.
  • Default is enabled
Show same sub-domain links
  • Same sub-doman (ssd) links go to other web pages in the same website.
  • When enabled ssd links are included in the list of links.
  • Default is enabled
Show same domain links
  • Same doman (sd) links go to other web pages in another part of the organizations website.
  • When enabled sd links are included in the list of links.
  • Default is enabled
Show external links
  • External links go to a different website.
  • When enabled external links are included in the list of links.
  • Default is enabled
Show non-html extension links
  • Some links can be identified as non-html links by looking at the file extension in the URL.
  • When file extensions correspond to common PDF, document or media types they will be identified as links to non-html documents.
  • Links to non-html documents will require testing for accessibility.
  • When enabled links with non-html file extensions are included in the list of links.
  • Default is enabled
